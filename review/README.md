# مشروع تصميم متكامل

## المكتبات المستخدمة
- Bootstrap 5
- Material Design
- Font Awesome 6
- Toastify JS
- SweetAlert2
- خط Cairo وTajawal

## متطلبات التثبيت
1. قم بتحميل جميع الملفات في مجلد المشروع
2. تأكد من وجود اتصال بالإنترنت لتحميل المكتبات الخارجية
3. افتح ملف index.html في المتصفح
